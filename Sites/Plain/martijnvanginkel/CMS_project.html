I heard from many people how annoying they find it to have to adjust something on their website. I therefore started a project to make a user-friendly cms with Laravel. In the cms you can change content such as photos, videos and text. In addition, posts can be created for the blog section of the website. For the design I have conducted usability tests with different people to ensure that the cms that I am creating is actually user-friendly. When I create a website I can link the content to the pages. This way I can make my own html pages myself and use this as a system to change the content. By doing this project I learned a lot about Laravel, learning from MVC, working with databases and how I can put all this online as a final product.
